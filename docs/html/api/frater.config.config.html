frater.config.config module¶

class Config(name='config', config_id=<factory>, meta=<factory>)[source]¶

Bases: dataclasses_json.api.DataClassJsonMixin

Base class for building configs in Frater. Used in frater.component, frater.stream and several other modules

Parameters

  • name (str) – name for the given config. Defaults to “config”

  • config_id (str) – config_id for the given config. Generally should let this stay default, unless deserializing an existing config. Will default to a random uuid4

  • meta (dict) – dictionary for storing extra metadata passed into config that is not an existing field for the class. If using the default constructor, then you just supply the dictionary. If using any of the constructors from json/yaml/dict, it will gather extra parameters for those fields.
